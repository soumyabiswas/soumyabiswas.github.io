E-Learning Forum

Make Learning easy, interactive and fun

E-Learning Forum is a web platform that allows users to collaborate with the tutors for effective learning, where users can post their doubts and get it solved by the experts.

With the outbreak of Corona, the traditional classroom based learning has become almost impossible, which laid to the basis of this online learning forum

Technologies used
  • React
  • Bootstrap

Key Features
  • Post a question and get it answered by experts
  • Experts post assignments for students.
  • Post Reference Material for students about the lessons like pdfs, videos, images.
  • Quiz for students assignment

In summary, what we built is:

  • E-Learning Forum(built in React) allowing students to interact with the experts after the classes, for getting their doubts cleared.
  • Easy way for all the teachers to share the reference material for students, that was used in the class to explain the concepts.
  • Better way for teachers to share the assignments.
  • Quiz for conducting tests for the students
  • Admin Dashboard for managing the student details, onboarding the students and other activities for managing the forum.
